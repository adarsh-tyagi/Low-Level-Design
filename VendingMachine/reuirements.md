# Requirements
1. Vending machine should support products with different price and quantities.
2. Machine should accept coins and notes of different denomination.
3. Machine should dispense selected product and return change if necessary.
4. Machine should keep track of available products and their quantities.
5. Should handle multiple transactions concurrently and ensure data consistency.
6. Should provide an interface for restocking products and collecting money.
7. Should handle exceptional scenarios like insufficient funds or out of stock products.
