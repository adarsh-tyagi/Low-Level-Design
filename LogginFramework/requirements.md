### Requirements
- Framework should support different log levels like DEBUG, INFO, WARNING, ERROR and FATAL.
- It should allow logging messages with a timestamp, log level and message content.
- Framework should support multiple output destinations like console and file.
- Should provide configuration mechanism to set up log level and output destination.
- Framework should be thread safe to handle the concurrent logging from multiple threads.
- Should be extensible to accomodate new log levels and output destinations in future.
